# DataLearner AI - 静态模板规格说明书

## 1. 项目概述

- **项目名称**：DataLearner AI 复刻版
- **类型**：静态单页HTML模板
- **核心功能**：AI模型选型工作台首页，展示排行榜、模型对比、API价格与教程资源
- **目标用户**：需要了解AI模型的开发者、研究人员和企业决策者

## 2. 视觉规格

### 布局结构
- 固定导航栏 + Hero区域 + 内容区块 + 底部页脚
- 最大内容宽度：1200px，居中显示
- 响应式设计，支持桌面端

### 色彩系统
- 主色：#2563EB (蓝色)
- 强调色：#3B82F6
- 背景色：#FFFFFF (亮色) / #0F172A (深色)
- 文字色：#1E293B (主文字) / #64748B (次文字)
- 边框色：#E2E8F0
- 卡片背景：#F8FAFC

### 字体
- 标题：system-ui, -apple-system, sans-serif
- 正文：system-ui, -apple-system, sans-serif
- 代码/数字：ui-monospace, monospace

### 组件样式
- 卡片：圆角8px，轻微阴影
- 按钮：圆角6px，hover状态色变
- 链接：蓝色带hover下划线
- 圆形数字标识：01/02/03步进

## 3. 页面结构

### 3.1 导航栏 (Header)
- Logo区域 (左侧)
- 导航菜单 (7项)：最新AI资讯、大模型排行榜、大模型评测基准、大模型列表、大模型对比、资源中心、工具
- 搜索框 (居右)
- 语言切换按钮：中文/English
- 主题切换按钮

### 3.2 Hero区域
- 标题：h1 "从评测到落地，更快选对 AI 模型"
- 描述文字
- 三个特性卡片：
  - 榜单 + 多维对比，快速锁定候选模型
  - API 价格 + 成本计算，选型前算清预算
  - 教程 + 术语 + 数据集，一站补齐背景知识
- 两个CTA按钮：开始模型选型、直接进入模型对比

### 3.3 模型选型三步走
- 步骤01：发现候选模型
- 步骤02：进入多维对比
- 步骤03：核算价格与预算
- 底部链接：大模型列表、成本计算器、部署教程

### 3.4 统计数据
- 834 收录大模型数量
- 225 原创 AI 博客数量
- 64 大模型评测数量

### 3.5 按任务开始 (四区块)
- 模型选型 → 先筛选候选模型
- 成本预算 → 先把价格和预算算清楚
- 榜单追踪 → 持续追踪模型变化
- 学习研究 → 了解背景知识和实践资料

### 3.6 精选推荐 (4卡片)
- 大模型评测对比
- 大模型 API 价格
- 部署与实践教程
- AI Agents 应用集

### 3.7 按类型浏览 (4分类)
- 榜单与评测
- 模型库与机构
- 数据与研究资料
- 内容与教程

### 3.8 核心工具 (5工具)
- 大模型对比工具
- API 价格页
- 成本计算器
- 工具导航
- Prompt 美化工具

### 3.9 页脚
- Logo + 简介
- 社交链接：X (Twitter)、Email
- 产品、资源、关于三个链接组
- 版权信息

## 4. 技术实现

- 单一HTML文件，内联CSS和少量JS
- 使用CSS Grid和Flexbox布局
- 模拟深色/亮色模式切换
- 模拟语言切换功能
- 所有交互为前端模拟，无后端
